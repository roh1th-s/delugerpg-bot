# DelugeBot

A simple python bot that automates certain aspects of [DelugeRPG](https://www.delugerpg.com/). 

- Uses selenium through `undetected-chromedriver` to make requests. Currently runs in non-headless mode, headless mode may trigger bot detection. 
- Captchas currently have to be manually solved if encountered. The bot will pause and beep for attention.

> **Disclaimer:** This was made for educational purposes only. Please refrain from using it maliciously.
